# Flipkart Login Page UI Clone

This project is a simple UI clone of the Flipkart login page built using HTML and CSS.

## Features
- Two-column layout  
- Styled login section  
- Input field for email/mobile number  
- "Request OTP" button  
- Terms of Use and Privacy Policy links  
- Create Account link  

## Concepts Practiced
- CSS Flexbox  
- Layout structuring  
- Typography  
- Button styling  
- Spacing and alignment  

## Description
The layout mimics the visual structure of Flipkart's login page for learning purposes only.  
No backend functionality is included.

## Technologies Used
- HTML5  
- CSS3  

## Purpose
This project was built as part of my frontend practice to improve UI layout and styling skills.
